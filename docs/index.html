The Runic Edda

Este es un starter project para gestionar y desplegar proyectos web minimalistas con Eleventy 3 y Github Pages.

  1. Edita _data/metadata.js con la información de tu proyecto y modifica la propiedad pathPrefix, con el nombre de tu repositorio en GitHub.

  2. Asegura que tu repositorio tenga activado GitHub Pages en la rama main, para desplegar la carpeta docs.

  3. Opcionalmente, edita eleventy.config.js con tus preferencias de configuración.

  4. Más información en la sección Acerca.

🎈🦡 Sitio hecho con Eleventy v3.1.1 🦡🎈
